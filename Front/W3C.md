# W3C

`2023-05-24`

> **World Wide Web Consortium**은 월드 와이드 웹을 위한 표준을 개발하고 장려하는 단체이다. <br>회원기구, 정직원, 공공기관이 협력하여 웹 표준을 개발하는 국제 컨소시엄이다.<br>
>
> > 설립취지로는 지속적인 성장을 도모하는 **프로토콜**과 **가이드라인**을 개발하여 **월드 와이드 웹**의 모든 잠재력을 이끌어 내는 것이다.
